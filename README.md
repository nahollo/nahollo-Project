# 규칙도둑 | Rule Thief

`규칙도둑`은 적이 가진 전투 규칙을 훔쳐 장착하고, 전장의 문법 자체를 바꿔가며 돌파하는 브라우저용 턴제 전술 로그라이크입니다.

핵심은 숫자 성장보다 `규칙 조작`입니다.

- 적은 단순 몬스터가 아니라 각자 시그니처 규칙을 가진 `규칙 보유자`
- 플레이어는 `스캔`, `절단`, `밀치기`로 전장을 읽고 탈취 조건을 만족시켜 `탈취 마크`를 새김
- 전투 후에는 훔친 규칙을 슬롯에 장착해 `패치 조합`을 만듦
- 모든 전투는 체인 안전 규칙 아래에서 결정론적으로 처리됨

## 기술 스택

## 조작

- `마우스 클릭`: 노드 선택, 액션 선택, 보드 타일/적 선택
- `WASD` 또는 `방향키`: 이동 모드에서 인접 타일 이동
- `1`: 공격
- `2`: 스캔
- `3`: 절단
- `4`: 밀치기
- `Space`: 대기
- `Esc`: 이동 모드로 복귀

## 구현 범위

현재 버전은 처음부터 끝까지 한 런을 플레이할 수 있는 `수직 슬라이스`입니다.

- 시작 메뉴 / 이어하기 / 코덱스 / 설정 / 크레딧
- 노드형 진행
- 일반 전투 / 휴식실 / 실험실
- 엘리트 전투 1종
- 최종 보스 1종
- 규칙 슬롯 / 백팩 / 패치 조합
- 탈취 마크 시스템
- 감사 열(Audit Heat)
- 전투 로그
- 로컬 저장

## 규칙 시스템 요약

룰은 항상 적용되는 패시브 법칙 조각입니다.

- `이동`: 후공이동, 위상보행, 잔상보행, 직선대시
- `교전`: 위치치환, 훅타격, 정면반격, 절단타
- `상태`: 독반전, 대기보호, 첫저주반사, 정화타
- `종결`: 시체벽, 시체폭발, 실행 캐시, 회수 잔해

패치 레시피 예시:

- `백도어`: 후공이동 + 위치치환
- `파편 바리케이드`: 시체벽 + 시체폭발
- `정적 순환`: 독반전 + 대기보호
- `유령 반격`: 위상보행 + 정면반격

## 전투 엔진 메모

전투는 9x9 격자 기반이며, 한 행동과 그 후속 효과는 모두 하나의 `Chain` 안에서 처리됩니다.

안전 장치:

- 같은 규칙 인스턴스는 한 체인에서 한 번만 발동
- 반응은 새 전체 반응 창을 열지 않음
- 체인 깊이 제한 `8`
- 제한 초과 시 `CHAIN LIMIT` 로그를 남기고 남은 후속 반응을 취소

테스트로 검증하는 항목:

- 반격 무한 루프 방지
- 연쇄 폭발 안전 종료
- 탈취 마크 획득
- 패치 조합 활성화
- 보스 3페이즈 전환
- 로컬 저장 무결성

## 앞으로 확장하기 좋은 방향

- 추가 캐릭터와 시작 규칙 분기

- 상점 / 오류구역 / 커널 룰
- 전투 연출용 사운드와 진동 피드백 확장
- 맵 분기와 감사 열 대응 이벤트 다양화
